# 🤖 3x-UI Telegram Bot

> **ربات تلگرام مدیریت پنل 3x-UI** — راهکاری جامع برای مدیریت کاربران، اشتراک‌ها و سرویس‌های VPN از طریق تلگرام

رباتی قدرتمند و کاربرپسند برای مدیریت پنل [3x-UI](https://github.com/MHSanaei/3x-ui) که به شما امکان می‌دهد بدون نیاز به ورود مستقیم به پنل، تمام کارهای مدیریتی را از طریق تلگرام انجام دهید. این ربات با پشتیبانی از سه سطح دسترسی (ابرمدیر، مدیر و کاربر عادی) نیازهای تمام نقش‌ها را پوشش می‌دهد.

## ✨ ویژگی‌های کلیدی

- 🔐 **سیستم احراز هویت چندسطحی** با سه نقش متمایز: ابرمدیر، مدیر و کاربر
- 👥 **مدیریت کامل کاربران** شامل ایجاد، ویرایش، حذف، تمدید و تعلیق اشتراک‌ها
- 📊 **داشبورد آماری** برای نمایش مصرف ترافیک، مدت زمان باقی‌مانده و وضعیت اتصال
- 🔄 **همگام‌سازی خودکار** با پنل 3x-UI از طریق API رسمی
- 💬 **اطلاع‌رسانی هوشمند** برای پایان اشتراک، مصرف ترافیک و رویدادهای مهم

## 🛡️ سطوح دسترسی

### ۱. ابرمدیر (Super Admin)

ابرمدیر بالاترین سطح دسترسی را دارد و می‌تواند تمام بخش‌های ربات را کنترل کند.

#### مدیریت مدیران
- ➕ افزودن مدیر جدید با شناسه تلگرام
- ❌ حذف مدیر از لیست
- 📋 مشاهده لیست کامل مدیران
- ✏️ ویرایش سطح دسترسی مدیران

#### مدیریت سرورها
- 🖥️ افزودن سرور 3x-UI جدید با اطلاعات اتصال (آدرس، پورت، یوزرنیم، پسورد)
- 🔗 اتصال و قطع اتصال سرورها
- 📈 نمایش وضعیت سرورها (آنلاین/آفلاین، مصرف CPU، RAM، ترافیک کل)
- 🗑️ حذف سرور از لیست

#### مدیریت تنظیمات ربات
- ⚙️ پیکربندی پیام خوش‌آمدگویی
- 💰 تنظیم قیمت‌ها و طرح‌های اشتراک
- 🌐 تغییر زبان پیش‌فرض ربات
- 📢 تنظیم کانال پشتیبانی و آیدی مدیر فروش
- 🔁 تنظیم زمان‌بندی نوتیفیکیشن‌ها (یادآوری تمدید، هشدار ترافیک)

#### گزارش‌گیری و تحلیل
- 📊 گزارش جامع درآمد روزانه، هفتگی و ماهانه
- 👥 آمار کاربران فعال، جدید و منقضی‌شده
- 💿 گزارش مصرف ترافیک کل سرورها
- 📈 نمودار رشد کاربران و درآمد
- 📤 خروجی گزارش در قالب Excel/CSV

#### مدیریت کاربران
- 🔍 جستجو و مشاهده اطلاعات کامل هر کاربر
- 🚫 مسدودسازی یا رفع مسدودیت کاربر
- 🎁 تمدید رایگان اشتراک کاربر
- 💵 ثبت پرداخت دستی برای کاربر
- 📝 ویرایش یادداشت کاربر

#### مدیریت کدهای تخفیف و آزمایشی
- 🎟️ ایجاد کد تخفیف با درصد/مبلغ مشخص و تاریخ انقضا
- 🆓 ساخت کد آزمایشی (Trial) با مدت زمان دلخواه
- 📋 مشاهده و مدیریت کدهای فعال
- 🗑️ حذف کدهای منقضی

#### پشتیبانی
- 💬 پاسخ به تیکت‌ها و پیام‌های پشتیبانی
- 📢 ارسال پیام همگانی به تمام کاربران یا گروهی خاص

### ۲. مدیر (Admin)

مدیر می‌تواند کارهای روزمره مدیریت کاربران و فروش را انجام دهد اما به تنظیمات حیاتی ربات دسترسی ندارد.

#### مدیریت کاربران
- 📋 مشاهده لیست کاربران با فیلتر (فعال، منقضی، مسدودشده)
- ➕ ایجاد اشتراک جدید برای کاربر (انتخاب پروتکل، مدت زمان، حجم)
- ✏️ ویرایش اشتراک کاربر (افزایش حجم، تمدید مدت)
- 🔗 ارسال مجدد لینک اشتراک و کد QR به کاربر
- ⏸️ تعلیق موقت یا فعال‌سازی مجدد اشتراک
- 🗑️ حذف کاربر از پنل 3x-UI

#### مدیریت فروش و پرداخت
- 💳 ثبت پرداخت کاربران و تأیید手动
- 📦 مشاهده طرح‌های اشتراک و قیمت‌ها
- 🎟️ اعمال کد تخفیف برای کاربر
- 💰 مشاهده درآمد شخصی و گزارش فروش

#### مدیریت کانفیگ‌ها
- ➕ ایجاد کانفیگ جدید (Vmess/Vless/Trojan/Shadowsocks)
- 🔍 جستجوی کانفیگ بر اساس UUID یا ایمیل
- 📊 مشاهده مصرف لحظه‌ای هر کانفیگ
- 🔄 ریست ترافیک کانفیگ

#### پشتیبانی کاربران
- 💬 پاسخ به پیام‌های کاربران
- 📌 انتقال تیکت به ابرمدیر در موارد پیچیده

#### گزارش‌گیری محدود
- 📊 گزارش فروش روزانه و ماهانه شخصی
- 👥 لیست کاربران تحت مدیریت

### ۳. کاربر عادی (User)

کاربر عادی می‌تواند اشتراک خود را مدیریت کرده، تمدید کند و از خدمات استفاده نماید.

#### ثبت‌نام و احراز هویت
- 📝 ثبت‌نام با شماره موبایل یا ایمیل
- 🔐 ورود با کد دعوت یا خرید مستقیم
- ✅ تأیید حساب کاربری

#### مدیریت اشتراک
- 📊 مشاهده اطلاعات اشتراک (مدت باقی‌مانده، حجم مصرف‌شده/باقی‌مانده)
- 🔗 دریافت لینک اشتراک و کد QR
- 📱 دریافت لینک‌های کانفیگ به‌صورت انفرادی
- 📈 نمودار مصرف روزانه ترافیک

#### خرید و تمدید
- 🛒 مشاهده طرح‌های اشتراک و قیمت‌ها
- 💳 پرداخت آنلاین و خرید اشتراک جدید
- 🔄 تمدید اشتراک فعلی قبل از انقضا
- ⬆️ ارتقا طرح اشتراک (افزایش حجم یا مدت)
- 🎟️ استفاده از کد تخفیف هنگام خرید
- 🆓 استفاده از کد آزمایشی (در صورت موجود بودن)

#### مدیریت حساب
- ✏️ ویرایش اطلاعات شخصی (نام، شماره تماس)
- 🔒 تغییر رمز عبور حساب کاربری
- 🌐 تغییر زبان ربات (فارسی/انگلیسی)
- 📱 اتصال/قطع اتصال دستگاه‌ها

#### پشتیبانی
- 💬 ارسال پیام به تیم پشتیبانی
- 📋 مشاهده تاریخچه تیکت‌ها
- ❓ دسترسی به سوالات متداول (FAQ)

#### نوتیفیکیشن‌ها
- ⏰ یادآوری تمدید قبل از پایان اشتراک (۳، ۱ و ۰ روز قبل)
- ⚠️ هشدار مصرف ۸۰٪ و ۱۰۰٪ ترافیک
- 📢 دریافت اطلاعیه‌ها و پیام‌های همگانی

## 🚀 نصب و راه‌اندازی

### ۱. کلون کردن مخزن

```bash
git clone https://github.com/your-username/3x-ui-telegram-bot.git
cd 3x-ui-telegram-bot
```

### ۲. ایجاد محیط مجازی

```bash
python3 -m venv venv
source venv/bin/activate   # Linux/macOS
# یا
venv\Scripts\activate      # Windows
```

### ۳. نصب وابستگی‌ها

```bash
pip install -r requirements.txt
```

### ۴. دریافت توکن ربات

1. در تلگرام به [@BotFather](https://t.me/BotFather) پیام دهید
2. دستور `/newbot` را ارسال کنید
3. نام و یوزرنیم ربات را وارد کنید
4. توکن دریافتی را ذخیره کنید

### ۵. پیکربندی فایل محیطی

```bash
cp .env.example .env
```

فایل `.env` را با ویرایشگر باز کرده و مقادیر را تکمیل کنید.

## ⚙️ پیکربندی

فایل `.env` شامل تنظیمات زیر است:

```env
# === تنظیمات ربات ===
BOT_TOKEN=your_telegram_bot_token_here
SUPER_ADMINS=123456789,987654321    # آیدی تلگرام ابرمدیرها (با کاما جدا شوند)
DEFAULT_LANGUAGE=fa                  # زبان پیش‌فرض: fa یا en

# === پایگاه داده ===
DB_HOST=localhost
DB_PORT=5432
DB_NAME=3xui_bot
DB_USER=postgres
DB_PASSWORD=your_db_password

# === Redis (اختیاری) ===
REDIS_URL=redis://localhost:6379/0

# === پنل 3x-UI ===
XUI_PANEL_URL=https://your-panel-domain:2053
XUI_USERNAME=admin
XUI_PASSWORD=your_panel_password

# === درگاه پرداخت ===
PAYMENT_GATEWAY=zarinpal            # zarinpal, nowpayment, etc.
PAYMENT_MERCHANT_ID=your_merchant_id
PAYMENT_CALLBACK_URL=https://your-domain.com/payment/callback

# === تنظیمات اشتراک ===
TRIAL_DURATION_DAYS=3               # مدت اشتراق آزمایشی به روز
TRAFFIC_WARNING_PERCENT=80          # درصد هشدار مصرف ترافیک
RENEWAL_REMINDER_DAYS=3,1,0         # روزهای یادآوری تمدید قبل از انقضا

# === تنظیمات نوتیفیکیشن ===
NOTIFICATION_CHECK_INTERVAL=3600    # فاصله زمانی بررسی (ثانیه)

# === پشتیبانی ===
SUPPORT_CHANNEL=@your_support_channel
ADMIN_CONTACT=@your_admin_username
```

### نمونه پیکربندی سرورهای متعدد

برای مدیریت چند سرور 3x-UI، پس از اجرای ربات از پنل ابرمدیر سرورها را اضافه کنید:

```
/add_server
> نام سرور: سرور آلمان
> آدرس: 185.x.x.x
> پورت: 2053
> یوزرنیم: admin
> پسورد: ********
```

## 🚢 استقرار

### استقرار با Docker (پیشنهادی)

```bash
# ساخت و اجرا با docker-compose
docker-compose up -d --build

# مشاهده لاگ‌ها
docker-compose logs -f bot

# توقف
docker-compose down
```

**docker-compose.yml:**

```yaml
version: '3.8'

services:
  bot:
    build: .
    container_name: xui-telegram-bot
    restart: unless-stopped
    env_file: .env
    volumes:
      - ./data:/app/data
      - ./logs:/app/logs
    depends_on:
      - db
      - redis
    networks:
      - xui-net

  db:
    image: postgres:15-alpine
    container_name: xui-postgres
    restart: unless-stopped
    environment:
      POSTGRES_DB: 3xui_bot
      POSTGRES_USER: postgres
      POSTGRES_PASSWORD: ${DB_PASSWORD}
    volumes:
      - postgres_data:/var/lib/postgresql/data
    ports:
      - "5432:5432"
    networks:
      - xui-net

  redis:
    image: redis:7-alpine
    container_name: xui-redis
    restart: unless-stopped
    ports:
      - "6379:6379"
    volumes:
      - redis_data:/data
    networks:
      - xui-net

volumes:
  postgres_data:
  redis_data:

networks:
  xui-net:
    driver: bridge
```

### استقرار دستی با Systemd

```bash
# کپی فایل سرویس
sudo cp deploy/xui-bot.service /etc/systemd/system/

# فعال‌سازی و اجرا
sudo systemctl daemon-reload
sudo systemctl enable xui-bot
sudo systemctl start xui-bot

# بررسی وضعیت
sudo systemctl status xui-bot
```

### استقرار روی سرور مجازی

```bash
# اجرای مستقیم با nohup
nohup python3 main.py > bot.log 2>&1 &

# یا با استفاده از screen
screen -S xui-bot
python3 main.py
# Ctrl+A ثم D برای خروج از screen
```
